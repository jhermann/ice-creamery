# Tropical Rum Punch (Deluxe)
<img style="float: right; margin-left: 1.5em;" width=240 alt="Logo" src="logo-Tropical_Rum_Punch.png" />

*Tropical Rum Punch* blends pineapple, orange, and
mango juices with a hint of lime for a bright, fruity base.

Coconut milk adds smoothness and depth, while a touch
of Jamaica Rum brings gentle warmth.
Pomegranate syrup rounds out the flavor
with a subtle tart sweetness.

The result is a refreshing sorbet that tastes like sunshine.<br clear=all />
